## 2. Systems of Equations

Given the system:

1) $2x + 3y = 12$  
2) $x - y = 1$

---

### 1) Methods and formulas
- Solve linear systems by substitution or elimination.
- Substitution: express one variable from one equation and substitute into the other.
- Elimination: multiply equations to cancel one variable when added/subtracted.

### 2) Step-by-step solution using substitution

From equation (2):
$$x - y = 1 \Rightarrow x = y + 1$$

Substitute into equation (1):
$$2(y + 1) + 3y = 12$$

Simplify:
$$2y + 2 + 3y = 12$$
$$5y + 2 = 12$$
$$5y = 10$$
$$y = 2$$

Now back-substitute into $x = y + 1$:
$$x = 2 + 1 = 3$$

### 3) Check the solution in both equations
- Eq (1): $2x + 3y = 2\cdot3 + 3\cdot2 = 6 + 6 = 12$ ✓
- Eq (2): $x - y = 3 - 2 = 1$ ✓

---

### 4) Final answer
- $x = 3$
- $y = 2$
