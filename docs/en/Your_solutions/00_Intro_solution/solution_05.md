# Section 0 — Mathematical Foundations
## 5. Trigonometry

Given:
- vector magnitude $|\vec{A}| = 15$
- angle with horizontal $\theta = 60^\circ$

Task: compute horizontal and vertical components of $\vec{A}$.

---

### 1) Formulas used
- For a vector in 2D with magnitude $A$ and angle $\theta$ from the x-axis:
  - $A_x = A \cos\theta$
  - $A_y = A \sin\theta$

### 2) Substitute values
- $A = 15$, $\theta = 60^\circ$.

- $\cos 60^\circ = \frac{1}{2}$
- $\sin 60^\circ = \frac{\sqrt{3}}{2}$

Then:
- $A_x = 15 \cdot \frac{1}{2} = 7.5$
- $A_y = 15 \cdot \frac{\sqrt{3}}{2} = \frac{15\sqrt{3}}{2} \approx 12.99$

---

### 3) Final answers
- Horizontal component: $A_x = 7.5$
- Vertical component: $A_y = \frac{15\sqrt{3}}{2} \approx 12.99$
