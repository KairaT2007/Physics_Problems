# Section 0 — Mathematical Foundations
## 3. Proportionality

Consider the Universal Law of Gravitation:

$$F = G \frac{m_1 m_2}{r^2}$$

Given: distance is doubled ($r \to 2r$), masses are halved ($m_1 \to \frac{1}{2} m_1$, $m_2 \to \frac{1}{2} m_2$).

---

### 1) Formulas used
- Universal gravitation law: $F = G \dfrac{m_1 m_2}{r^2}$.
- Factor change = new value / original value.

### 2) Substitute new quantities
- New mass product: $m_1' m_2' = \left(\frac{m_1}{2}\right)\left(\frac{m_2}{2}\right) = \frac{m_1 m_2}{4}$.
- New distance: $r' = 2r$, so $(r')^2 = (2r)^2 = 4r^2$.

Then new force:
$$F' = G \frac{m_1' m_2'}{(r')^2} = G \frac{\frac{m_1 m_2}{4}}{4r^2} = G \frac{m_1 m_2}{16 r^2}.$$

### 3) Change factor
- Factor = $\frac{F'}{F} = \frac{G \frac{m_1 m_2}{16 r^2}}{G \frac{m_1 m_2}{r^2}} = \frac{1}{16}$.

### 4) Interpretation
- Force decreases by factor $\frac{1}{16}$ (one sixteenth).
- Equivalent: force becomes 16 times smaller.

---

### 5) Final answer
- $F$ changes by factor $\frac{1}{16}$.
- New force $F' = \frac{F}{16}$.
